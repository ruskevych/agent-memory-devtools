# MVP Product Language

Agent Memory Devtools is a debuggable, correctable memory system for AI coding agents.

Emphasize:

- inspectable local memory
- replayable ingestion and retrieval decisions
- Memory Fix Mode feedback and rules
- Missing Memory Analysis suggestions
- confidence reports and conservative conflict detection
- SQLite storage and deterministic local embeddings

Do not claim hosted sync, auth, teams, enterprise policy, remote vector databases, or advanced embedding integrations are implemented.

Use consistent terminology: memory, feedback, rules, suggestions, confidence, conflicts, replay traces, Memory Fix Mode.
